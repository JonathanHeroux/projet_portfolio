Welcome to my portfolio. 
This website showcases my work as a full-stack web developer, including my projects, my articles and a admin panel to manage content dynamically.

Project Goals

- present my profile, skills, and professionnal background as a web developer.
- display all my projects
- display all my articles with comments allowed.
- provide an admin dashboard to add, edit, delete.

The website is build using a custom php MVC architecture with MySQL for database. Using Bootstrap and SCSS for the design.

Main Features

Public site

- Home page with introduction and call to action buttons.
- Projects page that loads all projects from the database from my website jonathanheroux.com.
- Articles page with dynamic content and comment section.
- About me page with an presentation of myself
- Contact section and page to contact me easily

  Admin panel

- Secure login system with role when subscribe.
- CRUD operations for: projects, articles, users.

Languages
- PHP
